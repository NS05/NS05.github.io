对于 Typora 写博客将更加容易

插入图片

接下来说明插入图片的操作，你当然可以手动的将图片放置在 <博客题目>这个文件夹中，再手动敲代码引用，但对于 Typora ，有更方便的方式。

打开 Typora 菜单栏 >>> 格式 >>> 图像 >>> 当插入本地图片时 >>> 复制图片到文件夹 >>> 选择<博客题目>这个文件夹

接下来，你在 Typora 编辑博客时，无需将图片放置于<博客题目>这个文件夹，而是任意的拖动图片到编辑区插入图片， Typora 将自动设置其路径及拷贝至<博客题目>这个文件夹，让你写博客更便捷。

至此，你的第一篇博客就创作完成啦，你可以 右键 MyBlog >>> 新建位于文件夹位置的终端窗口 >>> 逐次输入
