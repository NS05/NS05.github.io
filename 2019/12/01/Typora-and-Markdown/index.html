Typora and Markdown

发表于 2019-12-01 分类于 Pseudo Technical ， Hexo-Blog 阅读次数： Valine：

Hexo 写博客，使用的语言是 Markdown ，这种语言其实应用很广泛，Github 、简书等网站均有应用。之前没有接触过这种语言，但很容易上手，因为语法及其简单（不算扩展）。说个不恰当的比方吧， Markdown 之于 Html ，好比 Python 之于 C++ 。这么说很不恰当，别当真哈哈。每接触新一种语言，第一件事情就是选择合适的编辑器。之前说过， Sublime Text 是一个轻量级的编辑器，当然可以用安装插件的方式，用 Sublime Text 写Markdown，甚至于，安装的插件可以将你的代码编译，做到网页显示。但热衷于 Sublime Text 的我，却背叛了它，选择了 Typora。以下内容仅是个人使用的感受，我尽量尝试不写成广告。

Typora

前言

在我看来，无论你是热衷于写博客但对编程知之甚少的博主，又或是有文艺范爱写文章的程序员，这款软件均能满足你的需求。因为写代码与写文章终究是两回事，前者数字化、逻辑化，后者文字化、记叙化，这甚至是理与文、定量与定性、科学与哲学间的差别。回过头来，简而言之，你肯定不希望在你写博客时候，你的界面是这样的：
