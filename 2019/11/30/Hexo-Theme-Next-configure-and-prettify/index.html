Hexo-Theme Next configure and prettify

发表于 2019-11-30 分类于 Pseudo Technical ， Hexo-Blog 阅读次数： Valine：

Hexo 主题种类繁多，除了网上搜索之外（如github中Hexo主题），你当然可以自己创建自己的主题，参考Hexo主题官方文档，但显然，非Web前端工程师很难自创主题，并配置完善美化良好。当然，我也仅仅是个通信工程专业的Tyro，对简雅风格喜欢的我选择了 Next ，只可惜配置过程并不是很容易，这篇博客即是我对 Next 主题调整的过程的记述。<本文将一直更新>

Next 基本配置

Next 主题下载

Next 主题版本有许多，但请注意，Hexo 如果更新至 4.0.0 ，低版本的 Next 主题“上一页”“下一页”的按钮将会渲染失败，呈现出裸露的代码，据说需要将 Next 升级至 7.4.2 以上才可，故接下来的配置是对 Next 7.4.2 进行配置。首先你要有 Next 主题文件包，Don’t worry, it’s free. 现在就下载 >>> 点击这里

Source code (zip) 即是 Next 7.4.2 主题文件包，点击下载

因习惯，将下载下来的文件的文件名由 hexo-theme-next-7.4.2 改为 next ，将其放置到 MyBlog（你博客文件夹的名称，下文将均写为MyBlog） >>> themes 中，这样，准备工作就完成了
