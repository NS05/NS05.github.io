MacOS Set up Your Blog by [Hexo and Github]

NS05 · 2019-11-26 · 次阅读

环境配置

在MacOS系统中安装 node.js

下载安装包 macOS Installer 进行安装 >>> 立即前往

如遇到安装包来源于网络不明开发者，打开 系统偏好设置>>>安全性与隐私>>>通用>>>仍要打开 即可

打开 Terminal (终端)，输入sudo node -v检查 node 是否安装，可以查看 node 版本

同理，输入sudo npm -v检查 npm 是否安装，可以查看 npm 版本

其中 sudo 为获取 root 权限，Password 为你 mac 的用户密码，输入时不会显示哦

安装 git

下载 git 安装包进行安装 >>> 立即前往

如果下载器不启动，点击下图中 click here to download manually 并耐心等待

打开 Terminal (终端)，输入sudo git version检查 git 是否安装

安装 Hexo

打开 Terminal (终端)，输入sudo npm install -g hexo-cli

安装完成后，输入sudo hexo -v查看版本

Hexo在本地建立博客文档

Hexo建立的Blog文件夹

首先在你想建立 Blog 文件夹的位置建立文件夹，PS：升级到 macOS Catalina 后 Macintosh HD 文件夹是只读状态，需修改权限。方法：显示简介 >>> 共享与权限 >>> 右下角解锁 >>> 更改为读和写 >>> 锁定

右键该文件夹 >>> 新建位于文件夹位置的终端窗口 PS：如没有，则点击 访达（左上角） >>> 服务 >>> 服务偏好设置 >>> 勾选新建位于文件夹位置的终端窗口

在弹出的终端窗口逐次输入

sudo hexo i MyBlog // MyBlog 是你想要的文件名
cd MyBlog          // 进入 MyBlog 文件夹
sudo hexo g        // 编译
sudo hexo s        // 以4000端口预览

复制 http://localhost:4000 在 Safari （或其他浏览器）打开

注意：当想结束此进程，要用 Ctrl + C 结束，如果直接把终端关闭，再次运行时4000端口是被占用状态

需输入sudo lsof -i:4000来查看其pid号

再输入sudo kill -9 pid号来终止

这个即是我们的博客啦，如果将博客比作一棵树，那么 Hexo 已经帮助我们完成了枝干和根系，接下来我们要添加只是叶子。Hexo 创建的博客，默认主题叫做 landscape （位置：/MyBlog/themes）

Wait！接下来我们要让自己的博客能够在网上查阅（不要害怕，进行完下面的操作后仅能输入网址来访问，而不能被搜索引擎搜索到）

Github 托管个人博客

Github 端

github 是个深受程序员爱戴的开源网站，如果你接触过 github 下面的操作会相对便捷。

首先进入github官网，如果你尚没有账号需要先注册，点击右上角 sign up 注册。如果你已经拥有github账号，点击 sign in 登陆即可，接下来新建项目 repository

项目名为 用户名.github.io，按照如下选好，点击Create repository

接着找到你的项目并打开

点击 Settings

找到 Github Pages，选择 master branch

则我们看到，绿色框处即为我们的网址了，接下来配置我们的 hexo 端

Hexo 端

找到你博客的文件夹 MyBlog >>> _config.yml

打开此文件，我用的 Sublime Text 3 ，强烈安利此轻量级的写代码软件，当然你用Xcode等软件打开也可。这都不是重点，接下来我们找到 deploy ，修改成：

deploy:
    type: git
    repository: https://github.com/用户名/用户名.github.io.git
    branch: master

保存，请注意冒号 “:” 后有空格，且注意输入法为英文。

接下来同样，右键MyBlog（你的博客文件夹名称） >>> 新建位于文件夹位置的终端窗口 >>> 逐次输入

sudo npm install hexo-deployer-git --save  //安装 hexo-deployer-git 插件
sudo hexo clean                            //清除 hexo 上次的运行
sudo hexo g                                //编译
sudo hexo d                                //上传

如图则上传完成，上传完可能有几分钟延迟，接下来打开之前 github 绿色框处的网址

至此，我们的博客就能在网上访问啦。

接下来，我们要对我们的博客页面进行配置和美化，并尝试写自己的第一篇博客

首先选择自己的主题，当然如果你喜欢默认主题 landscape 也可以

这里放置一些外链资源：

Hexo官网 ： 便于官方文档查阅，查阅官方文档解决问题的官方手段

github中Hexo主题 ： 寻找你心爱的主题，当然你也可以去搜索引擎搜索

本站博客：

Hexo-Theme Next configure and prettify ： Next 主题配置与美化

Typora and Markdown ： Hexo 博客用 markdown 语法，软件推荐 Typora，当然 sublime text 也可

Write My First Blog ： 书写自己的第一篇博客
